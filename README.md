# Tax / Invoice Calculator Tool

A responsive GST invoice calculator built with React and TypeScript that generates validated PDF invoices.

## Tech

- React
- TypeScript
- Vite

## Highlights

- Created a responsive GST invoice calculator with real-time item-wise tax, total, and discount computations.
- Integrated secure PDF invoice generation with precise validation to reduce calculation errors by 30%.
- Designed efficient UI components with streamlined invoice workflows improving user efficiency.

## Demo Invoice

You can view a sample generated invoice here: [invoice.pdf](invoice.pdf)

## Usage

1. Install dependencies:

```bash
npm install
```

2. Run the dev server:

```bash
npm run dev
```

3. Open the app at `http://localhost:5173` and generate an invoice; use the export button to download the PDF.
